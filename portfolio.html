Sistema De Gestão De Estoque

DashBoard


  • Delegar: Onde o administrador delega quem pode fazer alterações no sistema
  • Relatório: Onde o administrador visualiza todos os movimentos realizados no sistema e quem os realizou
  • DashBoard: Onde encontram-se as estátisticas do estoque
  • Venda: Onde busca-se o produto com Id ou nome e registra-se como vendido
  • Produto: Lista dos produtos, onde os autorizados adicionam, editam e eliminam produtos
  • Sair: Para terminar a sessão

OBS: Essas funcionalidades só estão disponíveis na conta do administrador

Lista Dos Produtos


  • Novo Produto: Onde os autorizados podem cadastrar novos produtos
  • Ver: Este botão por sua vez permite a qualquer usuário a visualizar os detalhes do produto como preço, descrição e nome, ele também permite ver a quantidade em estoque e o Id mas só para as pessoas autorizadas
  • Editar: Onde os usuários que tenham sido delegar pelo administrador podem editar um produto ou qualquer um dos seus detalhes
  • Excluir: Onde os usuários autorizados por delegação podem excluir algum produto do estoque

Lista Dos Produtos


Esta é a parte visual da lista de produtos visualizada por uma pessoa não logada, que também espelha a visualização de uma pessoa não autorizada. Neste lado o botão 'Ver' apenas mostra a descrição, o preço e o nome do produto

Este projeto volta-se especificamente para a gestão de estoque, permitindo operações matemáticas automáticas para o preencimento do dashboard (Estatísticas)

Solta um alerta quando um determinado produto no estoque está em menor quantidade e registra Cada movimento realizado na aba 'Relatório' do administrador

Sistema De Login & Cadastro

Sistema De Login


Esta é a parte visual do sistema de login de uma aplicação web de uma organização prestadora de serviços

O presente sistema de login solicita que o usuário informe seu E-mail e palavra-passe no caso do usuário estiver registrado, caso não, por baixo há o link 'Cadastrar-se' para poder se cadastrar e solicitar serviços

A cima temos o link 'BACK' que nos leva de volta a página inicial

Sistema De Cadastro


Esta é a parte visual do sistema de cadastro da aplicação web da organização prestadora de serviços

O presente sistema de cadastro solicita que o usuário fornaça seus dados pessoais, de contacto e palavra-passe para o cadastro de usuário. Este sistema web, garante a segurança do usuário ao encriptar a palavra-passe do usuário não permitindo a sua visualização, nem pelo próprio administrador.

A cima temos o link 'BACK' que nos leva de volta a página de login e a baixo temos o botão cadastrar e cancelar

Sistema De Recolha e Exibição de Dados


O presente sistema limita-se em colectar os dados informados pelo usuário, cadastrar no banco de dados e em seguida exibir-los no lado em branco do sistema tal como a imagem espelha.

Aparente ser um sistema simples, porque é simples, mas de acordo com as necessidades do solicitador, acaba se tornando bem mais complexo devido a implementação de inúmeras funcionalidades

O sistema apresenta uma barra de navegação por cima que permite acessar diferentes partes do sistema
